# 📬 메일·공고 모니터링 프로그램 (Mail Monitor)

> **한 줄 소개:** 여러 정부지원사업 공고 사이트를 자동으로 돌아다니며, 우리 회사에 맞는 공고만 골라 이메일로 보내주는 프로그램이에요.
> **쉽게 말하면:** 흩어진 지원사업 공고를 매일 대신 뒤져서, 우리한테 쓸모 있는 것만 추려 메일로 보고해 주는 "공고 비서"예요.

---

## 🎯 이게 뭐예요?

정부·기관에서 올라오는 **지원사업 공고**(지원금, 바우처, 스마트공장, 수출지원 등)는 사이트마다 흩어져 있고 매일 새로 올라옵니다. 사람이 일일이 들어가 확인하기는 번거롭죠.

이 프로그램은 그 일을 **대신** 해줍니다.

1. 여러 공고 사이트(기업마당, K-Startup, 수출바우처 등)를 **자동으로 돌면서** 새 공고를 모읍니다.
2. 모은 공고 중 **우리 회사가 신청할 수 있는 것만** 걸러냅니다 (기본 기준: 인천 남동구 / 제조업).
3. 추려진 공고를 **이메일로 정리해서** 보내줍니다.

여기에 더해, 같은 폴더 안에 곁가지 기능 두 가지가 더 들어 있습니다 (아래 "이 폴더에 든 3가지 기능" 참고).

---

## ✨ 뭘 할 수 있어요?

이 폴더에는 서로 **독립적인 기능 3가지**가 함께 들어 있습니다.

| 기능 | 하는 일 | 어디서 도나요 |
|------|---------|--------------|
| **① 공고 수집·필터·메일** (핵심) | 여러 사이트의 지원사업 공고를 모아, 우리 회사에 맞는 것만 골라 메일로 보냄 | `monitor.py` |
| **② 정책자금 점검** | 소상공인시장진흥공단(소진공)의 정책자금 공고를 따로 점검해 보고서로 정리 | `loan/` 폴더 |
| **③ 고객사 서류 자동 입력** | 사업자등록증 등 서류 사진·PDF를 읽어 구글 시트에 자동 정리 (메일 기능과 완전 별개) | `customer_intake` |

추가로, 모아둔 공고와 설정을 **웹 화면(대시보드)** 에서 눈으로 확인할 수도 있어요 (`streamlit_app.py`).

## 🧰 시작하기 전에 (준비물)

이 프로그램을 쓰려면 컴퓨터에 이런 게 미리 있어야 해요.

- **Python (3.10 이상 권장)** — 프로그램을 돌리는 기본 엔진이에요. 깔려 있는지 확인하는 법:
  ```powershell
  python --version
  ```
  `Python 3.xx.x` 처럼 숫자가 뜨면 OK. "인식할 수 없다"가 뜨면 아직 안 깔린 거예요 → https://www.python.org 에서 설치 (설치 화면에서 **"Add Python to PATH"** 를 꼭 체크하세요).

- **구글(Gmail) 계정과 "앱 비밀번호"** — 공고를 메일로 보낼 때 쓰는 발송 계정이에요. 일반 로그인 비밀번호가 아니라 **앱 비밀번호**(구글 계정 보안 설정에서 따로 발급하는 16자리)가 필요합니다.

- **공공데이터(기업마당) API 키, Claude(Anthropic) API 키** — 공고를 가져오고(기업마당), 공고 내용을 짧게 요약(Claude)하는 데 씁니다.
  > ⚠️ 확인 필요: 두 키를 발급받는 정확한 사이트 주소는 폴더 안에서 확인되지 않았습니다. 처음 만들 때 어디서 받았는지 기억나는 곳에서 발급하세요. (기업마당 키 = `BIZINFO_API_KEY`, Claude 키 = `ANTHROPIC_API_KEY`)

- **인터넷 연결** — 공고 사이트를 돌아다니며 가져와야 하므로 필수예요.

## 📥 설치하기 (처음 한 번만)

1. **필요한 부품(라이브러리)들을 한 번에 설치합니다.** → 아래를 복사해 붙여넣고 Enter:
   ```powershell
   cd D:\mail
   python -m pip install -r requirements.txt
   ```
   > 💡 왜 하나요? 이 프로그램이 돌아가려면 같이 깔려 있어야 하는 다른 작은 프로그램들이 있어요. `requirements.txt`에 그 목록이 적혀 있고, 위 명령이 그걸 한꺼번에 깔아줍니다.

2. **비밀 설정값을 `.env` 파일에 채웁니다.** (메일 주소, 앱 비밀번호, API 키 등)
   - 폴더 안의 `.env.example` 파일을 복사해 이름을 `.env` 로 바꾼 뒤, 메모장으로 열어 값을 채웁니다.
   - 핵심 기능(공고 메일)에 꼭 필요한 값: `GMAIL_ADDRESS`, `GMAIL_APP_PASSWORD`, `BIZINFO_API_KEY`, `ANTHROPIC_API_KEY`
   > 💡 왜 하나요? 메일 비밀번호나 API 키를 코드에 직접 적으면 위험해요. 그래서 `.env`라는 별도 메모지에만 적어두고 프로그램이 거기서 몰래 읽어가게 합니다.
   > ⚠️ `.env` 파일은 절대 남에게 보내거나 깃허브에 올리지 마세요. 비밀번호가 담긴 파일이에요.

## ▶️ 실행하기

### 가장 중요한 점부터 (안전!)

이 프로그램은 두 가지 방식으로 실행돼요. **모드를 헷갈리면 실수로 진짜 메일이 나갈 수 있으니** 꼭 구분하세요.

- 🟢 **미리보기(안전) 모드** — 메일을 실제로 보내지 **않고**, "어떤 메일이 나갈지"만 보여줍니다. 평소 확인용은 항상 이걸 쓰세요.
  ```powershell
  cd D:\mail
  python monitor.py --dry-run
  ```
  **실행하면 이런 일이 일어나요:** 화면에 진행 로그가 흐르고, 수집한 공고/걸러낸 공고 요약이 `logs\` 폴더에 보고서로 저장됩니다. **메일은 나가지 않습니다.**

- 🔴 **진짜 발송 모드** — 골라낸 공고를 **실제로 이메일로 보냅니다.** 정말 보낼 때만 사용하세요.
  ```powershell
  cd D:\mail
  python monitor.py
  ```
  **실행하면 이런 일이 일어나요:** 위 과정과 같지만, 마지막에 **수신자에게 실제 메일이 발송됩니다.**
  > ⚠️ `--dry-run`을 빼고 그냥 `python monitor.py`만 치면 **진짜 메일이 나갑니다.** 헷갈리면 항상 `--dry-run`부터 돌려보세요.

### 웹 화면(대시보드)으로 보고 싶을 때

공고와 설정을 표·화면으로 편하게 보려면:
```powershell
cd D:\mail
python -m streamlit run streamlit_app.py
```
**실행하면 이런 일이 일어나요:** 잠시 뒤 인터넷 브라우저가 자동으로 열리고 `http://localhost:8501` 주소에서 관리 화면이 뜹니다. 창(검은 명령어 창)을 닫으면 화면도 꺼져요.

### 더블클릭으로 실행 (명령어가 익숙하지 않다면)

폴더 안에 클릭용 파일도 있어요.
- `run_monitor.bat` — 공고 모니터 실행 (※ 이건 **진짜 발송 모드**예요)
- `run_dashboard.bat` — 위 웹 대시보드 실행

## ❓ 자주 묻는 질문 (FAQ)

**Q. 이거 실행하면 진짜 메일이 나가나요?**
A. **명령에 따라 달라요.** `python monitor.py --dry-run`은 미리보기만 하고 메일을 안 보냅니다. `--dry-run` 없이 `python monitor.py`(또는 `run_monitor.bat` 더블클릭)는 **실제로 메일을 발송**합니다. 확인이 목적이면 항상 `--dry-run`을 붙이세요.

**Q. `python`이 인식 안 된대요 ("'python'은(는) ... 인식할 수 없습니다").**
A. 파이썬이 안 깔렸거나, 깔 때 "Add Python to PATH"를 체크 안 한 거예요. https://www.python.org 에서 다시 설치하면서 그 체크박스를 켜고, 검은 명령어 창을 새로 연 뒤 `python --version`으로 확인하세요.

**Q. `pip install`에서 빨간 글씨가 잔뜩 나와요.**
A. 보통 인터넷 연결 문제이거나 명령을 줄여 친 경우예요. `D:\mail` 폴더에서 정확히 `python -m pip install -r requirements.txt`로 다시 실행해 보세요.

**Q. `.env`가 뭐예요? 비밀번호는 어디에 넣어요?**
A. 메일 비밀번호·API 키 같은 비밀값을 적어두는 메모지 파일이에요. `D:\mail` 폴더의 `.env.example`을 복사해 `.env`로 이름을 바꾼 뒤 메모장으로 열고, `GMAIL_APP_PASSWORD=` 처럼 등호(=) 뒤에 값을 채우면 됩니다. (이 파일은 남에게 공유하면 안 돼요.)

**Q. 메일 비밀번호는 평소 쓰는 구글 로그인 비밀번호인가요?**
A. 아니에요. 구글 계정 보안 설정에서 따로 만드는 **"앱 비밀번호"(16자리)** 가 필요합니다. 일반 로그인 비밀번호로는 발송이 안 될 수 있어요.

**Q. 결과(보고서)는 어디에 저장돼요?**
A. 공고 모니터의 미리보기 보고서는 `D:\mail\logs\` 폴더에 저장됩니다. 정책자금 점검(②번) 보고서는 `D:\mail\reports\loan\semas_loan_scan.md`에 생깁니다.

**Q. 어떤 공고를 "우리 회사 것"으로 골라주나요?**
A. 기본 기준은 **인천 남동구 / 제조업**이에요. 베트남·동남아·해외·박람회·소상공인·지원금·스마트공장 관련은 잘 골라주고(가점), 단순 행정공지·교육·설명회·이미 마감된 공고·인천 남동구가 신청 불가인 공고는 자동으로 제외합니다.

**Q. 한 번 깔면 계속 쓸 수 있나요? 자동으로도 도나요?**
A. 설치는 처음 한 번만 하면 되고, 이후엔 실행 명령만 다시 치면 돼요. 평소에는 서버(Vercel)에 올려 정해진 시간마다 자동으로 도는 구조도 준비돼 있습니다.
> ⚠️ 확인 필요: 현재 자동 실행이 켜져 있는지, 몇 시에 도는지는 폴더만으로는 확정할 수 없습니다. 자동 실행을 멈추려면 서버(Vercel) 설정을 확인해야 합니다.

**Q. ②번 정책자금 점검은 어떻게 돌려요?**
A. 미리보기로 안전하게 돌리려면:
```powershell
cd D:\mail
python -m loan.semas.collector --run-mode dry-run --send-email false
```
결과 보고서는 `reports\loan\semas_loan_scan.md`에 생깁니다.

## 🛠️ 문제가 생겼을 때

- **증상:** 실행하자마자 `필수 환경변수 누락: GMAIL_ADDRESS`(또는 다른 이름)가 뜸 → **해결:** `.env` 파일이 없거나 그 값이 비어 있어서예요. `.env.example`을 복사해 `.env`로 만들고 빠진 값을 채우세요 (`GMAIL_ADDRESS`, `GMAIL_APP_PASSWORD`, `BIZINFO_API_KEY`, `ANTHROPIC_API_KEY`).
- **증상:** 메일이 안 나가거나 발송에서 오류 → **해결:** 구글 **앱 비밀번호**가 맞는지, `GMAIL_ADDRESS`가 올바른지 확인하세요. 또한 진짜 발송은 `--dry-run` 없이 실행했을 때만 됩니다.
- **증상:** 일부 사이트에서 공고가 안 모임 / "playwright 미설치" 경고 → **해결:** 일부 사이트는 추가 도구(Playwright)가 있어야 수집됩니다. 그 사이트만 건너뛸 뿐 전체 실행은 계속됩니다. 모든 사이트가 필요하면 Playwright 설치가 필요할 수 있어요.
  > ⚠️ 확인 필요: Playwright 설치 명령은 이 폴더에서 확정되지 않았습니다.
- **증상:** `streamlit`이 인식 안 됨 → **해결:** `streamlit` 대신 `python -m streamlit run streamlit_app.py` 형태로 실행하세요.

## 📂 결과물·파일은 어디에 있나요?

- **`D:\mail\logs\`** — 공고 모니터를 미리보기로 돌렸을 때 나오는 보고서가 저장되는 곳.
- **`D:\mail\reports\loan\semas_loan_scan.md`** — 정책자금 점검(②번) 결과 보고서.
- **`monitor.py`** — 이 프로그램의 "두뇌"(핵심 로직). 함부로 고치지 마세요.
- **`sites.json` / `groups.json` / `companies.json` / `settings.json`** — 어떤 사이트를 돌고, 누구에게 보내고, 어떤 회사 기준으로 거르는지를 정해둔 설정 파일들. 바꾸려면 이 파일을 메모장으로 신중히 수정하세요.
- ⚠️ **`.env`** — 메일 비밀번호·API 키가 담긴 비밀 파일. **남에게 공유 금지, 깃허브 업로드 금지.**
- ⚠️ **`seen_ids.json`** — 이미 한 번 처리·발송한 공고를 기억해 두는 파일(중복 발송 방지용). 지우면 예전 공고가 다시 메일로 갈 수 있으니 함부로 삭제하지 마세요.
